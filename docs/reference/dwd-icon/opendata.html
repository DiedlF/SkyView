Open Data Server DE

Free provision of spatial data of the DWD via the DWD's Open Data Server https://opendata.dwd.de

According to a change in the Deutscher Wetterdienst Act, which has come into effect on 25.7.17, the DWD has been given the legal mandate to make its weather and climate information available mostly free of charge. The DWD price list has been adapted accordingly and is available at www.dwd.de/preisliste. Please also see our conditions of use for further provisions.

Possibilities of access

The range of spatial data offered on the DWD's Open Data Server (https://opendata.dwd.de) is wide and includes model forecasts, radar data, current measurements and observations, a large amount of different types of climate data and many more.

The climate data of the DWD will continue to be provided on the server of DWD's Climate Data Center (CDC) at https://opendata.dwd.de/climate_environment/CDC/. Users wishing to access these through the Open Data Server will be forwarded from there via the server's climate subtree.

A number of data sets are additionally available from our GeoWebService (German language) portal in OGC compatible WMS and WFS formats.


When using the data server “opendata.dwd.de”, please note that service and availability levels cannot be guaranteed for this service.

For business or management processes that require higher data integrity and continuity, the responsible customer service can request the establishment of secure access via the geospatial data server service.

Data and provision services specifically certified for aviation are available exclusively from the Aviation Meteorology Department (luftfahrt@dwd.de).

More information

Use the server's weather subtree directory (https://opendata.dwd.de/weather/tree.html).

An overview of climate data offered is given separately and can be found on the CDC ftp server.

Information about adaptations, changes and extensions relating to the weather data available will be published in the Open Data Server news section. We invite you to subscribe to our Open Data RSS-news feeds to receive the news automatically.

Changes to the range of climate data offered will be listed in a log file available at https://opendata.dwd.de/climate_environment/CDC/Change_log_CDC_ftp.txt.
Users of climate data are recommended to subscribe to our CDC newsletter: https://www.dwd.de/DE/klimaumwelt/cdc/cdc.html.

For questions relating to climate data, please contact our climate data team by e-mail to klima.vertrieb@dwd.de.
